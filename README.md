# paint
一个基于SDK的画图小工具

- 可以选择画笔
- 画图形状包括直线、弧线、矩形、椭圆
- 可以选择画笔的虚实、粗细，以及画笔的颜色，包括图形的颜色填充
- 提供橡皮擦功能
